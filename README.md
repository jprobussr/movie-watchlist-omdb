# Movie Watchlist App

A movie search and watchlist application built with HTML, CSS, and JavaScript using the OMDb API.

## Features

- Search movies using the OMDb API
- Responsive grid layout
- Add movies to watchlist
- Persistent storage using localStorage
- Remove movies from watchlist
- Accessible focus states and clean UI design

## Tech Stack

- HTML5
- CSS3 (custom properties, responsive grid)
- Vanilla JavaScript
- OMDb API
- localStorage
